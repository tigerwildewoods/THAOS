# THAOS Install & Restore Guide

1. Extract this repository locally.
2. Replace placeholders in data/ with your real THAOS files if needed.
3. Push to GitHub main branch.
4. Enable GitHub Pages (Settings -> Pages -> Source: main / docs folder).
5. Use scripts/worldbackup.sh to create backups.
6. Use scripts/doink_run.sh to run ingestion pipeline locally.
